# ディレクトリマップ

## 1. 画面一覧

- メインウィンドウ (MainWindow)
- 設定画面 (SettingsWindow)

## 2. メインウィンドウの構成要素

### 2.1. メニューバー

- **ファイル(F)**
  - 新規作成(N)
  - 開く(O)
  - 保存(S)
  - 名前を付けて保存(A)
  - エクスポート(E)
    - HTML
    - PDF
    - Word (docx)
    - 画像 (PNG)
  - 終了(X)


- **設定(S)**
  - 設定(S)
- **ヘルプ(H)**
  - バージョン情報(A)

### 2.2. ツールバー

- CSS 選択プルダウン
- モード切り替え UI (ビューアーモード、Markdown 編集モード、リアルタイム編集モード)
- CSS編集パネル表示/非表示切り替えボタン



### 2.3. メイン表示・編集エリア

### 2.4. CSS編集パネル

- ツールバーのボタンによって表示/非表示が切り替わる。

## 3. 設定画面の構成要素

### 3.1. 操作ボタン

- **適用/保存ボタン**: 設定を保存し、設定画面を閉じる。
- **キャンセルボタン**: 設定を破棄し、設定画面を閉じる。
