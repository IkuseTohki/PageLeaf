# ディレクトリマップ

## 1. 画面一覧

- メインウィンドウ (MainWindow)
- 設定画面 (SettingsWindow)

## 2. メインウィンドウの構成要素

### 2.1. メニューバー

- **ファイル(F)**
  - 新規作成(N)
  - 開く(O)
  - 保存(S)
  - 名前を付けて保存(A)
  - エクスポート(E)
    - HTML
    - PDF
    - Word (docx)
    - 画像 (PNG)
  - 終了(X)
- **設定(S)**
  - 設定(S)
- **ヘルプ(H)**
  - バージョン情報(A)

### 2.2. ツールバー

- CSS 選択プルダウン
- モード切り替え UI (ビューアーモード、Markdown 編集モード、リアルタイム編集モード)
- CSS 編集パネル表示/非表示切り替えボタン

### 2.3. メイン表示・編集エリア

### 2.4. CSS 編集パネル

- ツールバーのボタンによって表示/非表示が切り替わる。
- **保存ボタン**: 編集したCSSを保存する。
- **タブコントロール**:
  - **全体タブ**:
    - 文字色
    - 背景色
  - **見出しタブ (H1-H6)**:
    - 文字色
    - 太字
    - 斜体
    - 下線
    - 取り消し線
    - フォントサイズ
    - フォントファミリー
    - 項番の自動採番
  - **引用タブ**:
    - 文字色
    - 背景色
    - 左枠線のスタイル（色、太さ、種類）
  - **リストタブ**:
    - マーカーの種類（黒丸、白丸、四角など）
    - インデント
  - **表タブ**:
    - 罫線のスタイル（色、太さ）
    - ヘッダーの背景色
    - セルのパディング
  - **コードブロックタブ**:
    - 背景色
    - 文字色
    - フォントファミリー

## 3. 設定画面の構成要素

### 3.1. 操作ボタン

- **適用/保存ボタン**: 設定を保存し、設定画面を閉じる。
- **キャンセルボタン**: 設定を破棄し、設定画面を閉じる。
