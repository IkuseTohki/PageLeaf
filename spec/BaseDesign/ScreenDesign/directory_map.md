# ディレクトリマップ

## 1. 画面一覧

- メインウィンドウ (MainWindow)
- 設定画面 (SettingsWindow)

## 2. メインウィンドウの構成要素

### 2.1. メニューバー

- **ファイル(F)**
  - 新規作成(N)
  - 開く(O)
  - フォルダを開く(F)
  - 保存(S)
  - 名前を付けて保存(A)
  - エクスポート(E)
    - HTML
    - PDF
    - Word (docx)
    - 画像 (PNG)
  - 終了(X)
- **編集(E)**
  - 切り取り(T)
  - コピー(C)
  - 貼り付け(P)
  - 元に戻す(U)
  - やり直し(R)
- **表示(V)**
  - モード
    - ビューアーモード
    - Markdown 編集モード
    - リアルタイム編集モード
  - CSS 選択 (プルダウンメニュー)
- **設定(S)**
  - 設定(S)
- **ヘルプ(H)**
  - バージョン情報(A)

### 2.2. ツールバー

- CSS 選択プルダウン
- モード切り替え UI (ビューアーモード、Markdown 編集モード、リアルタイム編集モード)

### 2.3. フォルダツリーエリア

- 各フォルダノードは展開・折りたたみが可能。

### 2.4. メイン表示・編集エリア

## 3. 設定画面の構成要素

### 3.1. フォルダツリー表示位置設定

- **表示位置選択**:
  - 左に表示 (ラジオボタン)
  - 右に表示 (ラジオボタン)

### 3.2. 操作ボタン

- **適用/保存ボタン**: 設定を保存し、設定画面を閉じる。
- **キャンセルボタン**: 設定を破棄し、設定画面を閉じる。
