引用デザイン案 v3 (アイコンスタイルの統一)

すべての「WITH ICON」パターンで、引用符の大きさ・位置・左パディングを共通化しました。

1. Simple (横ライン)

WITH ICON

アイコンとラインがセットになることで、引用ブロックとしての輪郭が最もはっきりと強調されるデザインです。

デザインシステム 2026
NO ICON

アイコンを外すと、文書の流れを妨げないミニマルな表現になります。

執筆ガイド

2. Card (カード型)

WITH ICON

カード型でもアイコン位置を統一。左側にアイコンを置くことで、テキストの読み始め位置が揃います。

UIパターン集
NO ICON

アイコンなしのカードは、よりフラットで落ち着いた、モダンなコンポーネントとして機能します。

コンポーネント設計

3. TopBottom (上下ライン)

WITH ICON

上下ラインでも左側にアイコンを配置。水平ラインと垂直方向のアイコンがバランス良く配置されます。

書籍レイアウトの原則
NO ICON

上下ラインのみにすると、非常にシンプルで、注釈や補足に近いニュアンスの引用になります。

エディトリアルデザイン

4. Modern (モダン)

WITH ICON

モダンデザインにアイコンを組み合わせた例です。斜体（Italic）を適用することで、アイコンの柔らかさと調和させています。

モダンな引用の解釈
NO ICON (Gradient Accent)

アイコンの代わりに、鮮やかなグラデーションバーと背後の柔らかな光を採用しました。引用符がなくても「特別なブロック」であることを主張します。
