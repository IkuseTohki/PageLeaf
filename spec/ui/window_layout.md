# UI 仕様：画面構成・レイアウト

## 1. メインウィンドウ

- **メニューバー**: ファイル操作、設定、ヘルプ（バージョン情報、チートシート）。
- **オーバーレイボタン**: 画面右上に配置された、表示モード切替および目次表示のためのフローティングボタン群。
- **メイン表示・編集エリア**: Markdown テキストの表示および編集を行う単一のコンポーネント。現在のモードに応じて表示内容と編集挙動が変化する。
- **サイドパネル**:
  - **CSS 編集パネル**: 画面右側上部に配置。CSS 選択、保存・リセットボタン、カテゴリ別タブを備える。
  - **フロントマター編集パネル**: 画面右側下部に配置。

## 2. 設定画面

- **目的**: アプリケーションの各種設定を行う。
- **設定項目**:
  - コードハイライト設定（テーマ、読み込み先）。
  - 画像貼り付け設定（保存先、テンプレート）。
  - エディタ設定（フォントサイズ、インデント）。
  - フロントマターテンプレート定義（プロパティ、初期値変数 `{Now}`, `{Date}`, `{Time}`）。

## 3. チートシートウィンドウ

- **目的**: ユーザー支援のためのリファレンス表示。
- **構成**: タブ切り替え式（Markdown / ショートカット）。
- **挙動**: モードレス表示。メインウィンドウとは独立して移動・リサイズが可能。再呼び出し時は Activate される。
