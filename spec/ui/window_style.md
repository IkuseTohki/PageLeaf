# StyleGuide: ウィンドウ・デザイン仕様

## 1. 共通の外観仕様

- **角丸 (Corner Radius)**: ウィンドウ本体、ボタン、入力フィールドは `4px` の角丸を適用する。
- **配色**: アプリケーションの基本テーマ（Light/Dark）の色定義を継承する。
- **アイコン**: タイトルバーおよびタスクバーには `Resources/AppIcon.xaml` で定義された標準アイコンを使用する。

## 2. モーダル・オーバーレイ（シャドウ）

モーダル表示時に背面に配置するレイヤーの仕様です。

- **背景色**: `Black` (#000000)
- **不透明度 (Opacity)**: `40%` (0.4)
- **重なり順 (Z-Order)**: メインウィンドウ内のすべてのコンテンツよりも手前。
- **アニメーション**:
  - 動作: フェードイン・フェードアウト（不透明度の変化）
  - 変化範囲: 0% ↔ 40%
  - 所要時間: 200ms
  - イージング: Ease-Out（開始は速く、終了に向けて減速する）

## 3. ウィンドウ出現アニメーション

- **モーダル**:
  - 種類: フェード + スケール
  - スケール変化: 95% -> 100%
  - 時間: 200ms
  - イージング: Ease-Out
- **モーダレス**:
  - 種類: フェードのみ
  - 時間: 200ms
  - イージング: Ease-Out

## 4. 配置規定

- **モーダル**: メインウィンドウの `中央`（CenterOwner）。
- **モーダレス**: メインウィンドウを完全に覆わない位置を初期位置とする。
