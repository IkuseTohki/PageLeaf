# UI 仕様：インタラクション・振る舞い

## 1. モーダルインタラクション

設定画面やチートシートなどのモーダルウィンドウ表示時の振る舞い。

- **シャドウ・オーバーレイ**: メインウィンドウ全体を半透明の黒色レイヤーで覆い、背面をブロックする。
- **オーバーレイの挙動**: フェードアニメーションを伴う表示・非表示。ウィンドウ外クリックで閉じる。
- **フォーカストラップ**: フォーカス移動をモーダルウィンドウ内に限定する。
- **出現アニメーション**: フェードインに加え、スケールやスライド等の動きを伴う。

## 2. 視覚的フィードバック (Dirty インジケータ)

ユーザーが変更を加えた際、視覚的に通知する。

- **エディタ側**: 画面右上のモード切替ボタンのアイコン左上に、オレンジ色のドットを表示する。
- **CSS パネル側**: 変更があったカテゴリのタブアイコン左上に、ドットを表示する。

## 3. ナビゲーション

- **目次 (TOC) ジャンプ**: 目次アイテムを選択すると、現在のモードに応じて適切な位置（編集モードなら該当行、ビューアーモードなら該当セクション）へスクロールする。
