# UI 仕様：インタラクション・振る舞い

## 1. モーダルインタラクション

設定画面やチートシートの表示時における、メインウィンドウとの連動挙動を定義する。

#### 視覚的効果とブロック

モーダル表示の際は、メインウィンドウ全体を半透明の黒色レイヤー（シャドウ・オーバーレイ）で覆い、背面への入力をブロックするものとする。オーバーレイはフェードアニメーションを伴って表示し、ウィンドウ外のクリックによってモーダルを閉じる挙動をサポートする。

#### フォーカスとアニメーション

フォーカス移動はモーダルウィンドウ内に限定（フォーカストラップ）し、アクセシビリティを確保する。ウィンドウの出現時は、フェードインに加えてスケールやスライド等の動きを組み合わせたアニメーションを適用する。

## 2. 視覚的フィードバック (Dirty インジケータ)

ユーザーが変更を加えた際、未保存状態を以下の箇所で通知するものとする。

- **エディタ側**: モード切替ボタンのアイコン左上にドットを表示する
- **CSS パネル側**: 変更のあったカテゴリタブのアイコン左上にドットを表示する

## 3. ナビゲーション

#### 目次 (TOC) ジャンプ

目次アイテムを選択した際、現在の表示モードに応じて適切な位置へスクロールさせるものとする。

- **編集モード**: 該当する行へジャンプする
- **ビューアーモード**: 該当するセクションへジャンプする
